# sentiment-analysis
Анализ эмоциональной тональности русскоязычных текстов с использованием цифровых методов в рамках гуманитарных исследований
Цель работы: Разработка алгоритма классификации эмоционально окрашенных высказываний в различных типах текстов с использованием методов машинного обучения.

Необходимо разработать полноценный пайплайн: сбор размеченных данных, предварительная обработка данных, обучение 3 предобученных моделей машинного обучения, применение ансамблирования моделей (рассмотреть различные подходы), подведение итогов и получение результативной модели
